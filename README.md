## 📝 소개
**NEWSEE**는 뉴스 기사 URL을 입력하면 AI가 자동으로 핵심 내용을 분석하고 요약해주는 크롬 익스텐션 서비스입니다. 복잡한 뉴스도 핵심 키워드, 감정 분석, 비판적 사고 질문 등 다양한 관점에서 쉽게 이해할 수 있도록 도와줍니다.

### 주요 기능
- 뉴스 URL 입력만으로 자동 기사 분석
- AI 기반 제목 재작성 및 요약 생성
- 키워드 추출 및 난이도 평가
- 감정/논조 분석
- 비판적 사고를 위한 질문 생성
- 관련 뉴스 자동 수집 및 보완 정보 제공

## 🤔 기술적 특징
- LangChain을 활용한 다중 AI 분석 체인 구성 (요약, 키워드, 감정, 난이도 등 병렬 처리)
- Cheerio 기반 웹 스크래핑으로 기사 본문 자동 추출
- 재시도(Exponential Backoff) 로직으로 안정적인 API 통신
- Recoil을 활용한 전역 상태 관리
- Next.js App Router 기반 서버/클라이언트 컴포넌트 분리
- Chrome Extension Side Panel UI 구성
